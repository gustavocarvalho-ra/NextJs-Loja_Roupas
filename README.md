## 🔧 Instalação
Execute os seguintes comandos no terminal da sua IDE:

1 - Clonar o repositório:
```
 git clone https://github.com/gustavocarvalho-ra/NextJs-Loja_Roupas.git
```
2 - Intalar as dependências do projeto:
```
🧰 pnpm i
```
3 - Iniciar aplicação:
```
💻 pnpm dev 
```
